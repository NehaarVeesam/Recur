Statement:
There is a biker going on a road trip. The road trip consists of n + 1 points at various altitudes. The biker starts his trip on point 0 with altitude equal 0.

You are given an integer array gain of length n where gain[i] is the net gain in altitude between points i​​​​​​ and i + 1 for all (0 <= i < n). Return the highest altitude of a point.

Approach: Prefix Sum
Time Complexity: O(n)
Space Complexity: O(1)
Calculate the prefix sum and on iterations find out the maximum value

class Solution:
    def largestAltitude(self, gain: List[int]) -> int:
        start = 0
        ans = 0
        for i in gain:
            start = start + i 
            ans = max(ans, start)
        return ans
